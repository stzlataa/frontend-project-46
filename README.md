## gendiff

Compares two configuration files (YAML or JSON) and shows a difference.

## Requirements

Node.js (>=13.2.0)

## Installation

```
npm link
```

## Usage

gendiff [options] <filepath1> <filepath2>

### Options

-V, --version        output the version number

-h, --help           output usage information

-f, --format <type>  output format
